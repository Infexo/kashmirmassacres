# kashmirmassacres
A chronological archive documenting documented massacres in Jammu & Kashmir from 1931 to present. This memorial website serves as a remembrance of the lives lost and aims to promote peace and reconciliation.

About

This project documents historical tragedies in Jammu & Kashmir region, preserving memory of victims and advocating for peace. The website features:


    Timeline of historical events

    Documented death tolls

    Source references

    Peace-oriented message


Tech Stack


    Pure HTML/CSS/JavaScript

    IBM Plex Sans & Playfair Display fonts

    No frameworks required


Usage

Simply open index.html in a web browser to view the website.

Sources

Data sourced from Wikipedia, declassified reports, human rights organizations, and survivor testimonies.
